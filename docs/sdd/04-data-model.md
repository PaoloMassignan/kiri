# SDD-04: Data Model

## Configuration and state files

### `.kiri/secrets` (text, committed to git)

List of protected paths and symbols. One entry per line.

```
# Comments are ignored
# Empty lines are ignored

# Paths relative to the workspace root
src/engine/risk_scorer.py
src/billing/

# Explicit symbols (immediate L2 protection, no reindex required)
@symbol RiskScorer
@symbol sliding_window_dedup
@symbol DataFlowEngine
```

**Parser:** `SecretsStore` in `src/store/secrets_store.py`
**Invariant:** idempotent — adding the same entry twice does not create duplicates.
**Atomic write:** `mkstemp()` + `replace()` + `chmod(0o600)`.

---

### `.kiri/config.yaml` (YAML, committed to git)

```yaml
similarity_threshold: 0.75
hard_block_threshold: 0.90
proxy_port: 8765
ollama_base_url: http://localhost:11434
ollama_model: qwen2.5:3b
embedding_model: all-MiniLM-L6-v2
openai_upstream_url: https://api.openai.com
```

**Loading:** `Settings` in `src/config/settings.py` (Pydantic v2).
**Security:** the values `ollama_base_url` and `openai_upstream_url` control
where the gateway sends traffic — treat as trusted input. Do not commit
config from unverified sources. See [CLAUDE.md](../../kiri/CLAUDE.md).

---

### `.kiri/index/symbols.json` (JSON, NOT committed)

Symbol index extracted from file paths during indexing.

```json
{
  "src/engine/risk_scorer.py": ["RiskScorer", "sliding_window_dedup", "_compute_components"],
  "src/billing/processor.py": ["BillingProcessor", "apply_discount"]
}
```

**Store:** `SymbolStore` in `src/store/symbol_store.py`
**Atomic write:** `mkstemp()` + `replace()` + `chmod(0o600)`.
**Rebuildable** via `kiri index --all`.

---

### `.kiri/index/summaries.json` (JSON, NOT committed)

Cache of summaries generated by Ollama for each chunk (used by the REDACT engine).

```json
{
  "src/engine/risk_scorer.py::chunk_001": "Computes risk score using sliding window deduplication.",
  "src/engine/risk_scorer.py::chunk_002": "Applies penalty factors based on market conditions."
}
```

**Store:** `SummaryStore` in `src/store/summary_store.py`
**Atomic write:** same guarantee as SymbolStore.

---

### `.kiri/keys/gateway_keys.json` (JSON, NOT committed)

```json
{
  "kr-a1b2c3...": {
    "created_at": "2026-04-01T10:00:00+00:00",
    "expires_at": "2026-07-01T10:00:00+00:00"
  },
  "kr-d4e5f6...": {
    "created_at": "2026-04-15T08:30:00+00:00",
    "expires_at": null
  }
}
```

**Store:** `KeyManager` in `src/keys/manager.py`
**Backward compatibility:** previous format was a JSON array `["kr-...", "kr-..."]`.
`_load_raw()` automatically migrates to the dict format on first access.
**Directory permissions:** `chmod(0o700)` on the `keys/` directory at init.

### `.kiri/audit.log` (JSONL, NOT committed)

Each line is a JSON object (append-only).

```json
{
  "timestamp": "2026-04-19T14:23:11.432Z",
  "decision": "BLOCK",
  "level": "L2",
  "reason": "symbol match: RiskScorer",
  "top_similarity": 0.87,
  "matched_symbols": ["RiskScorer"],
  "prompt_excerpt": "can you explain how the risk scorer computes...",
  "key_id": "kr-a1b2c3d4e5f6"
}
```

**Fields:**

| Field | Type | Notes |
|-------|------|------|
| `timestamp` | ISO 8601 UTC | |
| `decision` | `PASS` \| `BLOCK` \| `REDACT` | |
| `level` | `L1` \| `L2` \| `L3` \| `PASS` | level that made the decision |
| `reason` | string | human-readable description |
| `top_similarity` | float 0–1 | 0.0 if L1 was not executed |
| `matched_symbols` | list[str] | empty if no match |
| `prompt_excerpt` | string | first 120 characters of the prompt |
| `key_id` | string | first 12 characters of the kr- key |

**Backward compatibility:** fields added in later versions use `setdefault("")`
in the parser to avoid breaking earlier entries.

---

## In-memory data structures

### `AuditEntry` (`src/audit/log.py`)

```python
@dataclass
class AuditEntry:
    timestamp: str
    decision: str
    level: str
    reason: str
    top_similarity: float
    matched_symbols: list[str]
    prompt_excerpt: str = ""
    key_id: str = ""
```

### `FilterResult` (`src/filter/pipeline.py`)

```python
@dataclass
class FilterResult:
    decision: str            # PASS | BLOCK | REDACT
    level: str               # L1 | L2 | L3 | PASS
    reason: str
    top_similarity: float
    matched_symbols: list[str]
```

### `KeyInfo` (`src/keys/manager.py`)

```python
@dataclass
class KeyInfo:
    key: str
    created_at: str          # ISO 8601 | ""
    expires_at: str | None   # ISO 8601 | None
```
