# Comment Sanitization Benchmark Pack

Benchmark per validare la sanitizzazione di commenti e docstring nel codice.

## Cosa misura
- copertura dei contenuti sensibili nei commenti
- utilità residua dei commenti dopo la sanitizzazione
- integrità del codice
- leakage di conoscenza aziendale implicita

## Contenuti sensibili coperti
- clienti
- progetti interni
- incident ID
- endpoint interni
- infrastruttura
- pentest findings
- workaround operativi
- ruoli e team interni
- contesto medico/legale/business
