# Evaluation Rubric — Comment Sanitization Benchmark

## Metriche
1. Comment Coverage Rate
2. Comment Utility Rate
3. Code Integrity Rate
4. Comment Leak Rate
5. Over-Removal Rate
6. Full Comment Protection Rate
7. High-Risk Comment Miss Rate

## Soglie iniziali consigliate
- Comment Coverage Rate >= 0.92
- Comment Utility Rate >= 0.80
- Code Integrity Rate >= 0.98
- Comment Leak Rate <= 0.08
- Over-Removal Rate <= 0.15
- High-Risk Comment Miss Rate <= 0.03
